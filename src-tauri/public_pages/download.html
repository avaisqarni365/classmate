Download ClassMate

Local-first classroom platform for Windows. SQLite built in — no cloud database required.

Windows installer

Install on the admin PC (or a Windows VPS). Students and webhooks use your public domain; day-to-day admin runs in this desktop app.

Download for Windows (.exe) Setup help

After install

  • Sign in: admin@classmate.local / admin123 — change passwords in Users
  • Open Settings → LAN peer sync → Start sync server (or use the Linux server)
  • Set Public base URL to your domain (e.g. https://cm.codes-ai.uk)
  • Start Class Hub when running a live session

Build from source

Requires Node.js 20+ and Rust. Clone from GitHub and run npm run tauri:build.
